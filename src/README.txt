README file for TANK-OFF! game

To launch:
	1. Extract contents of ZIP file
	2. Open index.html in a web browser

System Requirements:
	Browser:
		- Mozilla Firefox release after 2016
		- Google Chrome
		- Microsoft Edge Chromium
		- Other modern browsers should work but are untested
	
	Inputs:
		- Game Controller with 2 analogue sticks required
		- Xbox One controller has been tested
		- Mouse and Keyboard is NOT supported
